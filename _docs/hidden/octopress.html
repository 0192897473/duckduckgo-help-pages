To use DuckDuckGo for your Octopress site's search, follow these instructions:

  1. Change the simple-search URL in the YAML configuration to "http://www.duckduckgo.com/”.
  2. Open the navigation partial that is under “source -> _includes” and replace the inputs in the fieldset with this code:
  

  {% if site.simple_search %}
<form action="{{ site.simple_search }}" method="get">
  <fieldset role="search">
    <input type="hidden" name="sites" value="duckduckgo.com" />
    <input class="search" type="text" name="q" results="0" placeholder="Search&hellip;"/>
  </fieldset>
</form>
  {% endif %}
Don't forget to replace duckduckgo.com with the name of your site.
