How to set up DuckDuckGo in KDE Plasma

 
  1. Go to: Configure Desktop -> Account Details -> Web Shortcuts, scroll down to DuckDuckGo, activate it in the third column and apply.
  2. Then start Krunner (Alt + F2), type “dd:your search term” (without “”) and the DuckDuckGo query will be opened in your favorite browser

If you are a user of KDE browsers (rekonq or Konqueror), DDG will also be in the list of your default search engines by activating it as described above.
