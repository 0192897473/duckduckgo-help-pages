--- title: IP Blocking category: Hidden order: 239 ---

To combat spamming from botnets and other malicious services, our system automatically blocks IP addresses that approach a queries-per-second limit. If you've been mistakenly blocked, via the use of our API or other means, you should see service returned within a few hours. If, after 24 hours, you're still unable to access DuckDuckGo, please send an email to ops@duckduckgo.com and we can investigate further.
