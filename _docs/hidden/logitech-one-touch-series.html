--- title: Logitech One-Touch Series category: Hidden order: 105 ---

Some Logitech devices have a feature called One-Touch Search, which allows you to easily use a search engine through a special button.

Follow these instructions to use this feature with DuckDuckGo:

  1. Open up the device's configuration software.
  2. Go to the "One-Touch Search" option.
  3. Select "Custom URL".
  4. Type in "https://duckduckgo.com/?q=%search%".
