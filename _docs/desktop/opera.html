Install our extension (recommended)

  • ✔ Adds DuckDuckGo's instant answers to Google and Bing search results
  • ✔ Adds a toolbar button that lets you search with DuckDuckGo
  • ✗ Does not add DuckDuckGo as a Speed Dial entry (can be done manually)

Making DuckDuckGo the default search engine

  1. Select Opera > Preferences (on Mac) or Opera > Options (on Windows)
  2. Under Search click the drop down and select DuckDuckGo

Making DuckDuckGo the homepage

  1. Select Opera > Preferences (on Mac) or Opera > Options (on Windows)
  2. Under On startup select Open a specific page or set of pages
  3. Enter https://duckduckgo.com

Adding DuckDuckGo as a Speed Dial entry

  1. Visit the DuckDuckGo homepage
  2. Right click anywhere on the homepage
  3. Select Add to Speed Dial
  4. Alternatively, you can add a Speed Dial link to random instant answers using this extension.

Uninstalling a DuckDuckGo extension

  1. Select Opera > Extensions
  2. Find the DuckDuckGo extension in the displayed list and click the X

Changing to a different default search engine

  1. Select Opera > Preferences (on Mac) or Opera > Options (on Windows)
  2. Under Search click the drop down and select your new desired default search engine

Removing DuckDuckGo as the homepage

  1. Select Opera > Preferences (on Mac) or Opera > Options (on Windows)
  2. Under On startup select Open a specific page or set of pages
  3. Find DuckDuckGo in the displayed list and click the X

Removing DuckDuckGo as a Speed Dial entry

  1. Right click on the DuckDuckGo Speed Dial entry
  2. Select Delete
