Set DuckDuckGo as the default search engine

  1. Click the ... icon at the top right
  2. Click Settings
  3. Scroll down and click View advanced settings
  4. Under Search in the address bar with select Add New
  5. Select DuckDuckGo and click Add as default!

Set DuckDuckGo as the homepage

  1. Click the ... icon at the top right
  2. Click Settings
  3. Under Open with select A specific page or pages
  4. Select Custom and enter https://duckduckgo.com in the input field
  5. Click +
